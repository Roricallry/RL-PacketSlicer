
##### 1. 网络利用率
各节点启动网络利用率探针：
```sh
python3 util_collector.py --port 500【节点序号】 --name 【节点名称】
```

节点Node2启动网络利用率探测器：
```sh
python3 util_monitor.py
```

发起一次群体探测，需执行：
```sh
curl -X GET http://192.168.2.2:5003/metrics
```


##### 2. 网络负载控制器
节点Node7启动接收服务器：
```sh
iperf3 -s -p 5201 -i 1   # -i 1 表示每秒输出一次报告
```
启动负载控制器：
```sh
python3 load_test.py 
```
启动负载测试：
```sh
curl -X POST http://192.168.2.139:5060/start
```
停止负载测试：
```sh
curl -X POST http://<发送端节点IP>:5060/stop
```

##### 3. ICMP网络时延探测
节点Node2启动ICMP报文时延探测器：
```sh
python3 util_latency.py --port 5010 --name Node2
```
发起一次探测请求：
```sh
# 测量到 8.8.8.8 的延迟，持续 30 秒（默认）
curl "http://192.168.2.2:5010/measure?target=192.168.2.5"
# 测量到 1.1.1.1 的延迟，持续 10 秒
curl "http://192.168.2.2:5010/measure?target=192.168.2.5&duration=10"
```

##### 4. 信息熵计算
启动信息熵计算器：
```sh
sudo /home/abc/PacketSlicer/venv/bin/python3 entropy_probe1.py --iface br0 --name Node3
```
测量单台设备（默认 camera，时长 30 秒）
```bash
curl "http://192.168.2.21:5011/measure"
```

指定设备 bulb，时长 60 秒
```sh
curl "http://192.168.2.21:5011/measure?device=bulb&duration=60"
```

同时测量所有三台设备，时长 15 秒
```sh
curl "http://192.168.2.21:5011/measure?device=all&duration=15"
```


```sh
kill -9 [PID]
```
